# WareHub CRUD API

## Description
This project is a **Warehouse Management System (WMS)** with CRUD functionality (Create, Read, Update, Delete) for products, orders, and stock management.  
It uses:

- **FastAPI** for API endpoints.
- **PostgreSQL** for data storage.
- **Docker & Docker Compose** for easy deployment and execution.

The system includes features such as:

- Creating new products with name, description, price, and stock quantity.
- Reading all products.
- Updating product details (price, stock, description).
- Deleting products.
- Searching products based on criteria (e.g., name, availability).
- Managing stock levels (increase/decrease quantity).
- Product statistics (e.g., total products, average cost).

---

## Requirements

- Docker 20+
- Docker Compose 1.29+
- Python 3.12+ (for development without Docker)
